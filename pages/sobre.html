Sobre este portifólio

  • Inspiração

Este portifólio teve como inspiração a incrível Live Coding oferecida pela DIO. Foi apresentado e desenvolvido um site sobre o multiverso do Spider-man, apenas com html, css e javascript puro.

A anfitriã da Live Coding foi a Juliana da DIO.

Como a Live Coding foi muito inspiradora, e sugerido pelos autores para fazermos alterações, então fiz as minhas:


  •    Mudei o tema e as cores.
